Games and Apps

Here's where you can try out some of our web apps and games we've created!

Hot or Cold Game

This game asks for a number. If the number is close to an internally- generated secret number, it tells you the guess is hot or cold. Many aspects of JavaScript were used in this project, built on the jQuery library.

Shopping List App

This app allows you to add items to a list and mark them off once you've done them! You can also remove all items at once. This is a mobile-first webb-app utilizing the jQuery library.
